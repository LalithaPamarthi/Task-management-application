# TaskFlow - Task Management Application

A modern full-stack task management web application built using React.js, Node.js, and MongoDB.

## Features

* User Authentication
* Create, Update, Delete Tasks
* Task Priorities and Categories
* Responsive Dashboard
* Modern UI Design

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Tailwind CSS
